Description

Read coverage

Methods

Input reads: HiFi and ONT reads (guppy 6.3.7~6.3.8)
Mapper: Winnowmap v2.03 release (--MD -W repetitive_k15.txt -ax [map-pb | map-ont])
Filter: Primary read alignments extracted with samtools view -F0x104

Display Conventions and Configuration

Pink: HiFi Primary alignments.
Blue: ONT Primary alignments.

Data access

Original bam files are available on AWS GenomeArk.
