Description

This track represents the alignments of the PacBio IsoSeq data against the individual assemblies of the T2T Primates project. These IsoSeq alignments were used for annotation of novel isoforms and novel genes.

Display Conventions and Configuration

This track follows the display conventions for bam alignment tracks. More information about each gene can be found by clicking on the specific transcript model. The following color key is used:

  •     - Protein Coding Genes

Methods

This tracks displays alignments generated by using minimap2 to map PacBio IsoSeq long reads to the primate genome assemblies.

    minimap2

    IsoSeq fastq files were mapped with Minimap2 with parameters -ax splice:hq -uf -t 28. This allows to generate splice-aware alignment of PacBio Iso-Seq or Nanopore cDNA or Direct RNA reads against a reference genome.
