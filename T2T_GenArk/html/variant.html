Description

Short-read variant detection

Method

For the T2T references, we generated karyotype-specific references following a published masking method (Rhie et al. 2023, DOI: https://www.nature.com/articles/s41586-023-06457-y) to improve variant representation in sex chromosome pseudoautosomal regions (PARs). Reads from XX and XY samples were aligned to their respective masked T2T reference genomes. We used GATK v4.4.0.0 HaplotypeCaller for initial variant calling, followed by GenotypeGVCFs for joint genotyping, and applied following quality control to SNPs and indels.

Variant filteration criteria:

For SNPs: -filter "QD < 2.0" -filter "QUAL < 30.0" -filter "SOR > 3.0" -filter "FS > 60.0" -filter "MQ < 40.0"

For indels: -filter "QD < 2.0" -filter "QUAL < 30.0" -filter "FS > 200.0".

Variants with a genotype quality below 20 were further removed to enhance the accuracy of our variant call set.

Within-sample haplotype phasing

Following variant calling and subsequent QC, we applied statistical phasing using BEAGLE 5.4 without a reference panel. Specifically we used the default 3 burnin iterations and 12 subsequent phasing iterations in BEAGLE. For the detailed pipeline supporting the phasing of these primate genomes https://github.com/Lqh09/T2T-Great-Ape-Diversity/tree/main/phasing
