Description

This set of tracks provides per base conservation scores and conserved elements using PhastCons (Siepel et al., 2005).

Display Conventions and Configuration

No colors specified.

Methods

    Multiple Sequence Alignment (MSA)

    Alignments were generated using wfmash all-to-all, producing one PAF with each assembly as a target / reference. Next, wgatools was used to filter PAF (16 PAFs) for min alignment len >10 Mbp and then convert PAF>MAF (384 MAFs).

    Conservation scores and conserved elements (CEs)

    Tracks have been produced according to PhastCons HOWTO similarly to similarly to as in Secomandi et al. 2023 using both default parameters (unsupervised EM learning) and a 4-fold degenerate site model. The 4d model was further refined by fine tuning of the model parameters --target-coverage and --expected-length maximizing enrichment (Jaccard/IoU) in CHM13 exon sequences. Code is available in the associated Github. The initial model estimated with ML and a tree from TimeTree. For each chromosome, alignments were split in 1 Mbp chunks, and predictions were combined.

    Notes

    Additional data is available at s3://garrisonlab/t2t-primates/PHAST/all-vs-all/
