Description

This track shows the locations of known quality and accuracy issues in the HG002v1.1 genome assembly.

These issues are catalogued and described in the issues section of our "HG002-issues" github site.

Display Conventions and Configuration

The names of entries in this track include the description and genomic locations of the issues as they appear on the github site.

Methods

The locations and descriptions of currently known HG002v1.1 issues were downloaded from https://github.com/marbl/HG002-issues/issues using the "hub" github API, called by a python script.
