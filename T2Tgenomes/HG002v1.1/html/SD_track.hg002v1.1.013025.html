Description

A track showing segmental duplications as generated by sedef on the assembly: HG002v1.1 Column descriptions can be found here: columns.

Display Conventions and Configuration

This section describes track configuration controls, or any special display conventions such as the meaning of different colors in your tracks.

The colours used for level of similarity segmentation standard:

less than 90% similarityPurple
90 - 98% similarityLight to dark gray
98 - 99% similarityYellow
greater than 99% similarityOrange

Methods

Sedef was run with default parameters on a RepeatMasked genome assembly. The resulting output was then converted into a browser friendly format (bed9 + extra fields).
