Description

These tracks display nucleotide frequencies generated by Mitchell Vollger's NucFreq program run on Winnowmap2 primary alignments of HiFi Revio reads called with DeepConsensus v1.2 or the ONT Epi2Me release of "Q28" reads v1.1 HG002 assembly. The input bam files are available here and here.

Display Conventions and Configuration

The counts displayed in black are counts of the most common base call in reads aligned at a given position. The counts in red are for the second most common base call in reads aligned at the same position.
